# Lere Skincare — Luxury Landing Page

A modern, elegant landing page designed for a premium skincare brand.  
This project focuses on visual storytelling, refined UI, and a smooth user experience that reflects a high-end identity.

---

## ✨ Overview

Lere is a conceptual skincare brand created to showcase my ability to design and develop visually compelling, conversion-focused landing pages.

The goal was to create a digital experience that feels:
- minimal yet luxurious  
- feminine yet powerful  
- modern yet timeless  

---

## 🎯 Features

- Split-screen hero (light & dark contrast)
- Smooth scroll reveal animations (IntersectionObserver)
- Fully responsive design (mobile-first)
- Elegant typography and spacing system
- Section-based storytelling layout

---

## 🛠 Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, responsive design)
- JavaScript (IntersectionObserver for animations)

---

## 🎨 Design Approach

This project was designed with a strong focus on:

- Visual hierarchy and balance  
- Premium color palette (soft neutrals + deep plum tones)  
- Clean, intentional spacing  
- Subtle interactions to enhance user experience  

Every section is structured to guide the user naturally through the brand story.

---

## 📱 Responsiveness

The layout is fully optimized for:

- Mobile devices  
- Tablets  
- Desktop screens  

Ensuring a consistent and polished experience across all screen sizes.
